Capture Task

Use the Capture Task to create 2D snapshots of your 3D scene.

The Capture Task is only available if Voyager Story is started in QC mode.
  1. Select the Capture Task in the task bar.
  2. Drag the view (orientation: left mouse button, pan: right mouse button, zoom: scroll wheel) until you’re happy with the scene view.
  3. Click Take to take a snapshot. The snapshot appears in the task panel.
  4. Click Save to save the snapshot back to the server.
  5. Click Save in the top right to save the scene.

You can use the Download button next to the Save button to save a local copy of the captured image.
